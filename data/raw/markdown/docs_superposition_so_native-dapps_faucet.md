#### 
[](https://docs.superposition.so/native-dapps/faucet#token-output-every-5-hours)

  * **Superposition Testnet Native tokens:** 0.5 SPN, 0.5 wSPN, 100 CAT and 0.5 fUSDC
  * **Arbitrum Sepolia Tokens:** 0.5 SPN
